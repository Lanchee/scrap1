======
scrap baidu port
======



Overview
========

Scrapy is a fast high-level port scanning tool, by using uslookup
to get the ip address from DNS, we try connect every port of the getting ip.

Requirements
============

* Python 2.7 or Python 3.3+
* Works on Linux, Windows, Mac OSX, BSD

using
=======

The quick way::
    python main.py
